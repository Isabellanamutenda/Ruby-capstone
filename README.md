# Ruby-capstone


## Built With
- Ruby
- Rubocop
- RSpec


## Getting Started

- git clone https://github.com/Isabellanamutenda/Ruby-capstone.git
- cd Ruby-capstone
- Run bundle insatall
- Run ruby main.rb file
- To test run rspec spec for all tests
- To run tests for a specific file run rspec/ spec/<name of the file.rb>
